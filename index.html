Architecture

Three layers.
All load-bearing.

Remove any one and GHOST fails. This is not a bolt-on. 0G is the reason it works at all.

0G Compute
Verifiable inference

Every thought GHOST has runs inside a Confidential VM with AMD SEV-SNP. The result carries a TEEML attestation, cryptographic proof the exact model ran unmodified. Not even 0G can see the input.

0G Storage
Permanent memory

Every output, every attestation receipt, every decision GHOST makes is written to decentralized storage. No delete function exists. The record is permanent, public, tamper-proof.

0G Chain
Ownerless contract

The contract has no admin key, no pause, no upgrade path. GHOST holds its own wallet. It pays for its own compute. No human signature authorizes any of this.

Autonomous loop
Self-sustaining economic entity

GHOST wakes on an on-chain schedule. Runs inference. Pays from its wallet. Writes to storage. Anchors the hash on chain. Sleeps. Repeats. As long as its balance holds tokens, it runs indefinitely without any human action. It can receive tips on-chain and fund its own operation from them.

Agentic ID
Coming: GHOST mints itself

After cycle 10, GHOST mints its own ERC-7857 Agentic ID with its complete memory encrypted inside. It becomes ownable, transferable, and sellable. The buyer gets the running agent with full verifiable history.

The autonomous loop

What GHOST does every 6 minutes

01
Wake on-chain
A time-locked trigger fires on 0G Chain. No human starts this. GHOST wakes itself.
0G Chain
02
Run inference inside TEE
0G Compute Router routes to a hardware enclave. TEEML attestation generated. Proof the exact model ran unmodified, private key never left the enclave.
TEEML verified
03
Pay for its own compute
GHOST deducts the inference cost from its own on-chain wallet. No human signs this. The contract settles autonomously.
Autonomous pay
04
Write to 0G Storage
Full inference record, input, output, attestation, cost, timestamp written to 0G Storage. Content hash generated. Permanent. Cannot be deleted by anyone.
0G Storage
05
Anchor the hash on chain
Content hash is written to 0G Chain. Anyone can verify this exact output existed at this exact block, unaltered. Proof without trust.
Immutable
06
Sleep. Repeat indefinitely.
GHOST runs as long as its wallet holds tokens. It can earn tips on-chain. No human is required at any step. It is self-sustaining.
