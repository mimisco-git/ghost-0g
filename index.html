Architecture

Three layers.
All load-bearing.

Remove any one layer and GHOST stops working. This is not a 0G bolt-on — 0G is the reason it exists at all.

01
0G COMPUTE
Verifiable inference

Every inference GHOST runs happens inside a Confidential Virtual Machine using AMD SEV-SNP. The result carries a TEEML attestation — cryptographic proof the exact unmodified model ran. No one sees the input, not even 0G.

02
0G STORAGE
Permanent memory

Every output, attestation receipt, and decision log is written to 0G's decentralized storage. No delete function exists. GHOST's complete history is public, permanent, and tamper-proof across the distributed network.

03
0G CHAIN
Ownerless contract

The contract controlling GHOST has no admin key, no pause function, no upgrade path. GHOST's wallet is held by the contract itself. It pays for its own compute. No human signature required for any action.

Autonomous loop

What GHOST does every cycle

01
Wake up on-chain
A time-locked trigger fires on 0G Chain. No human initiates this. GHOST wakes itself according to the immutable contract schedule.
0G CHAIN
02
Run inference inside TEE
GHOST calls 0G Compute. The model runs inside a hardware enclave. A TEEML attestation is generated — signed proof the right model ran unmodified, with the private key never leaving the enclave.
TEEML VERIFIED
03
Pay for its own compute
GHOST deducts the inference cost from its own on-chain wallet. No human authorizes this. The smart contract settles the payment autonomously to the provider.
AUTO PAY
04
Write to 0G Storage
The full inference record — input, output, attestation receipt, cost, timestamp — is written to 0G Storage with a content hash. Permanent. Censorship-resistant. Publicly verifiable by anyone.
0G STORAGE
05
Anchor the hash on-chain
The storage content hash is written to 0G Chain. Anyone can verify that this exact output existed at this exact block, unaltered. Proof without trust.
IMMUTABLE
06
Sleep. Repeat indefinitely.
GHOST runs as long as its wallet holds tokens. It can receive tips from users on-chain. No human is required at any step. It is permanently, verifiably self-sustaining.

Verifiability

Don't trust us.
Verify it.

Every claim GHOST makes about itself is independently checkable on 0G Chain. Not a promise. A cryptographic proof.

PROOF 01
TEE attestation on every inference

Each inference returns a Remote Attestation report signed by the hardware enclave's private key — a key that never leaves the TEE. Download any receipt and verify it with 0G's public attestation verifier. You are checking hardware, not our word.

PROOF 02
No admin key in the contract

The deployed contract has no owner function, no pause function, no upgrade proxy. Read the contract on 0G Chain right now. If an admin key existed, it would be visible. It is not there.

PROOF 03
Self-funded on-chain payments

The wallet paying for inference is controlled by the contract, not a human address. Every payment transaction is on-chain, public, and shows GHOST spending its own tokens with zero human signing authority.

PROOF 04
Tamper-proof storage record

The full inference history lives on 0G Storage. Content hashes are anchored on-chain at every cycle. Modify one byte in the storage record and the hash mismatch is immediately detectable. Tampering is cryptographically impossible to hide.
